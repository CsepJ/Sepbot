You should change the bot's prefix

First, Go to Config.js file.
```json
{

"prefix" : "셉봇",

}
```
Change it
```json
{

"prefix" : "Sepbot",

}
```


Then, You should change sending message too.

```javascript
message.channel.send("Hi");
message.reply("Hi");
```
```javascript
message.channel.send("hello");
message.reply("this is example");
```
